# 习刻 (Habitick)

![Kotlin](https://img.shields.io/badge/Kotlin-1.9.0-blue.svg) ![Compose](https://img.shields.io/badge/Jetpack%20Compose-Material3-green.svg) ![Room](https://img.shields.io/badge/Database-Room-orange.svg)

**习刻 (Habitick)** 是一款基于 **Jetpack Compose** (Material3) 开发的现代化 Android 习惯追踪应用。注重极简设计与强大的数据可视化，支持多种习惯类型与灵活的统计分析。

## ✨ 核心功能

* **多类型支持**：普通打卡、数值记录（如喝水）、计时习惯（如冥想）、时刻记录（如起床）。
* **可视化统计**：
    * 📅 **日历与热力图**：月度日历视图与年度 GitHub 风格热力图。
    * 📊 **多维图表**：打卡频次柱状图、数值趋势折线图。
    * 🏷️ **标签分析**：蓝色调饼图展示打卡标签分布。
    * 🔥 **最佳连续记录**：自动计算并高亮显示历史最佳连续打卡天数。
* **交互与管理**：
    * **拖拽排序**：长按列表即可自定义习惯排序。
    * **标签与备注**：打卡时可添加标签和心得备注。
    * **历史补卡**：支持修改历史日期的打卡状态与内容。
* **数据备份**：支持 CSV 格式的数据导入与导出。
